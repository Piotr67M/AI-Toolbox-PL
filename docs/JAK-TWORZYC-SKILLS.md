# Jak tworzyć własne Skills

Każdy Skill powinien mieć osobny folder i plik `SKILL.md`.

Minimalna struktura:

```text
skills/nazwa-skilla/
└── SKILL.md
```

Dobry Skill opisuje: kiedy go używać, potrzebne wejście, kolejność pracy, format wyniku, kontrolę jakości i ograniczenia. Nazwę folderu zapisuj małymi literami z łącznikami. Instrukcje powinny być krótkie, jednoznaczne i możliwe do sprawdzenia.
