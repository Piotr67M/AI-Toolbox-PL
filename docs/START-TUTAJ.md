# Start tutaj

## Najprostszy sposób użycia

1. Opisz cel oraz oczekiwany wynik.
2. Wybierz właściwy Skill z `skills/`.
3. Dołącz tylko niezbędne, dozwolone dane.
4. Poproś AI o zaznaczenie założeń, braków i niepewności.
5. Sprawdź wynik przed wysłaniem lub podjęciem decyzji.

Przykład: „Użyj instrukcji z `skills/meeting-assistant/SKILL.md`. Na podstawie poniższych anonimowych notatek przygotuj protokół według `templates/protokol-spotkania.md`.”

## GitHub

Po rozpakowaniu ZIP możesz przeciągnąć zawartość folderu `AI-Toolbox` przez **Add file → Upload files**, a następnie wybrać **Commit changes**.
