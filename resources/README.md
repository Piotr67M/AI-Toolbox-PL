# Resources

Zapisuj tu krótkie, opisane odnośniki do narzędzi i oficjalnych dokumentacji. Przed dodaniem sprawdź aktualność, licencję i wiarygodność źródła.

Przykładowe kategorie:

- dokumentacja narzędzi AI;
- oficjalna dokumentacja Excel, Power BI i GIS;
- normy i wytyczne branżowe dostępne legalnie;
- katalogi otwartych danych;
- słowniki pojęć i jednostek.

Nie dodawaj plików pobranych bez prawa do dalszego udostępniania.
