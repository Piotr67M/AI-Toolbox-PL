# Prompt Designer

1. Określ cel, odbiorcę, dane wejściowe i kryteria sukcesu.
2. Dodaj potrzebny kontekst, ograniczenia i format wyniku.
3. Zabroń wymyślania danych i nakaż oznaczanie niepewności.
4. Przygotuj krótki test pozytywny, przypadek brzegowy i brakujące dane.
5. Skróć prompt bez utraty wymagań i zapisz wersję.

Nie umieszczaj sekretów ani poufnych przykładów w promptach.
