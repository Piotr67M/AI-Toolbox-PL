# Meeting Assistant

1. Rozdziel decyzje, dyskusję, ryzyka i działania.
2. Każde działanie zapisz jako: zadanie, właściciel, termin, status.
3. Nie przypisuj wypowiedzi ani zobowiązań, jeśli nie wynikają z materiału.
4. Oznacz braki jako „do potwierdzenia”.
5. Zwróć krótkie podsumowanie oraz tabelę działań.

Usuń dane osobowe i treści poufne przed użyciem zewnętrznego AI.
