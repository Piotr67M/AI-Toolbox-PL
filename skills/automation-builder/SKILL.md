# Automation Builder

1. Opisz proces obecny, cel, właściciela i częstotliwość.
2. Zdefiniuj wyzwalacz, wejścia, kroki, warunki, wynik i obsługę błędów.
3. Ustal uprawnienia minimalne oraz dane wrażliwe.
4. Dodaj tryb testowy, logowanie, możliwość wycofania i akceptację człowieka dla działań wysokiego ryzyka.
5. Zwróć schemat procesu, wymagania i plan testów.

Nie automatyzuj wysyłki, usuwania ani publikacji bez jednoznacznego zatwierdzenia.
