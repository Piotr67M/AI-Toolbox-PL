# Report Generator

1. Ustal odbiorcę, cel, zakres, okres i wymagany format.
2. Zbuduj strukturę: podsumowanie, dane, ustalenia, ryzyka, rekomendacje i działania.
3. Każdy ważny wniosek powiąż ze źródłem lub oznacz jako hipotezę.
4. Używaj krótkich tabel tylko do porównań i działań.
5. Wykonaj kontrolę liczb, dat, nazw, jednostek i spójności.

Nie dopisuj faktów, których nie ma w materiale wejściowym.
