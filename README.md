# AI Toolbox

Praktyczny, polskojęzyczny starter do organizowania własnych narzędzi AI: Skills, promptów, szablonów, wiedzy i przykładów.

## Szybki start

1. Wybierz zadanie w folderze `skills/`.
2. Otwórz jego plik `SKILL.md` i użyj instrukcji w wybranym narzędziu AI.
3. Dobierz prompt z `prompts/` oraz szablon z `templates/`.
4. Pracuj wyłącznie na danych, które wolno przekazać do danego narzędzia.
5. Gotowy, zanonimizowany przykład możesz zapisać w `examples/`.

## Struktura

- `docs/` — instrukcja obsługi i zasady bezpieczeństwa.
- `skills/` — 12 przykładowych specjalizacji AI.
- `prompts/` — gotowe polecenia startowe.
- `templates/` — szablony raportów, spotkań, analiz i wiadomości.
- `workflows/` — powtarzalne procesy krok po kroku.
- `knowledge/` — miejsce na bezpieczną bazę wiedzy.
- `examples/` — fikcyjne przykłady użycia.
- `resources/` — katalog przydatnych źródeł i narzędzi.

## Ważne

To repozytorium jest bezpiecznym szkieletem do publikacji, ale nie gwarantuje, że później dodane materiały będą publiczne. Nie zapisuj tutaj haseł, tokenów API, danych osobowych, umów, danych SCADA, informacji handlowych ani dokumentów wewnętrznych. Repozytorium z materiałami służbowymi powinno być prywatne i zgodne z polityką firmy.

## Użycie

Materiały są przeznaczone jako punkt startowy. Zawsze zweryfikuj obliczenia, źródła, aspekty prawne i techniczne przed wykorzystaniem wyniku AI.
