# Baza wiedzy

Umieszczaj tu wyłącznie materiały, które wolno przechowywać w tym repozytorium. W publicznej wersji najlepiej dodawać własne notatki ogólne i linki do legalnie dostępnych źródeł.

Proponowany podział: `wind/`, `bess/`, `gis/`, `data/`, `ai/`. Nie kopiuj całych chronionych publikacji. Dla każdego materiału zapisz źródło, datę, właściciela praw i zakres zastosowania.
